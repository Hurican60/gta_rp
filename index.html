⚙ Configuration Supabase
Pour que tes sauvegardes fonctionnent partout, tu dois connecter ta base de données Supabase.
C'est gratuit et ne prend que 2 minutes.

1. Va sur supabase.com → crée un compte gratuit
2. Crée un nouveau projet (attends ~1 minute)
3. Va dans SQL Editor et colle le contenu de setup.sql (fourni avec ce fichier)
4. Va dans Settings → API et copie l'URL et la clé anon public
5. Colle-les ci-dessous
